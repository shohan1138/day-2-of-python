# Day 2 of Python

This repository contains Python scripts created on the second day of learning Python. The files demonstrate basic programming concepts such as arithmetic operations, input handling, and simple calculators.

## Files Description

### `39=12.py`
A simple program that adds the digits of a two-digit number.  
**Example:** If the input is `35`, the output will be `3 + 5 = 8`.

### `BMI_Calculator.py`
Calculates Body Mass Index (BMI) based on user input for height and weight. Demonstrates arithmetic operations and user input handling.

### `Tip_calcilator.py`
A simple tip calculator that helps users split bills and calculate tips. Useful for practicing float operations and input/output formatting.

### `days_to_live_untill_90.py`
Estimates how many days, weeks, and months a person has left until age 90 based on their current age. Great for practicing math and time calculations.

### `len.py`
Demonstrates the use of the `len()` function to calculate the length of a string.
